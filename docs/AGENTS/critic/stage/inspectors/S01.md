## 3. 内容审查要点

- [ ] 主题定义是否清晰、不模糊？
- [ ] 领域地图是否覆盖了相关的主要子领域？
- [ ] 核心关键词列表是否可用于有效搜索？
- [ ] 时间线是否涵盖了该领域的关键里程碑？

## 4. 常见错误模式

- 主题定义过宽或过窄
- 遗漏了重要的子领域
- 关键词过于通用，无法有效搜索
