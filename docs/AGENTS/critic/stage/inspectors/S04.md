## 3. 内容审查要点

- [ ] 核心假设是否以 '如果...那么...' 形式陈述？
- [ ] 每个假设是否有明确的可测量预测？
- [ ] 是否考虑了零假设（Null Hypothesis）？
- [ ] 假设之间是否有清晰的层次关系？
- [ ] 假设是否可证伪？

---

## 4. 常见错误模式

- 假设过于模糊，无法测量
- 假设不可证伪
- 与文献矛盾但未说明
- 预测不明确
