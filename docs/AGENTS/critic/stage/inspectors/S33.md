## 4. 常见错误模式

- S28-S31 被跳过，S33 直接替代生成（严重错误：S28-S31 是强制前置阶段）

- 图表文件路径错误（引用了不存在的图片）

- 未使用项目已配置的 venue 模板
- refs.bib 中缺少正文引用的条目（orphan cite）
- Anti-Leakage Prompt 未应用
- 数值与原始实验数据不一致
- **学术风格退化**：论文出现口语化表达、博客式标题、contractions、bullet list 正文
