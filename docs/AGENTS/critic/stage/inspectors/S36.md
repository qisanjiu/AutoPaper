## 4. 常见错误模式

- 遗漏了审稿意见未回应

- 引入新问题（如语法错误、逻辑矛盾）
- 未遵循 Accept/Revert 规则，强行接受评分下降的修改
- 迭代次数超限仍未停止
- 自动检查未执行或失败被忽略
